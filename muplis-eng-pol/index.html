La Muplis is a fast tool to search sentenes from Tatoeba.org database in Polish and English.

- Can I use it if I lose internet access?
- Yes, if this happens try opening this very address from this browser.
It should still open. A latest version of Chrome, Safari or Firefox is required.
To increase the overall quality of the sentences displayed this app
1. includes only sentences owned by tatoeba.org users,
2. excludes sentences tagged as "needs native check".
