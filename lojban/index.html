la sutysisku

la sutysisku is a fast dictionary for the Lojban language.

- Can I use it if I lose internet access?
- Yes, if this happens try opening this very address from this browser.
It should still open. A latest version of Chrome, Safari or Firefox is required.
  • 'search' prefers official words and shows them first, 'search+' sets equal priority to both official and experimental words.
  • Press / to focus the search field.
  • For advanced users: to enable regular expressions search in titles, definitions and notes either start the string with ^ or end with $: ^ui, sku$ are examples.
Transliterations to zbalermorna orthography are provided for your searches.
